Estrutura da tabela contas_mes
CREATE TABLE contas (
    id INT(11) NOT NULL AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci NOT NULL,
    valor DECIMAL(10,2) NOT NULL,
    tipo ENUM('fixa', 'variavel') CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci NOT NULL,
    vencimento DATE NOT NULL,
    paga TINYINT(1) NULL DEFAULT 0,
    porcentagem DECIMAL(5,2) NULL DEFAULT NULL,
    dias_restantes INT(11) NULL DEFAULT NULL,
    categoria VARCHAR(20) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci NOT NULL DEFAULT 'pessoal'
);


Observações importantes
- nome → descrição da conta (água, luz, celular, etc.).
- valor → pode ser atualizado quando mudar (fixas mudam anualmente, variáveis mensalmente).
- tipo → ajuda a diferenciar despesas fixas e variáveis.
- vencimento → data limite para pagamento.
- paga → controle simples (0 = não paga, 1 = paga).
- porcentagem e dias_restantes → não precisam ser digitados manualmente; podem ser calculados via consulta SQL ou no PHP.
Exemplo de cálculo
- Porcentagem:
SELECT nome, valor, 
       (valor / (SELECT SUM(valor) FROM contas_mes)) * 100 AS porcentagem
FROM contas_mes;
- Dias restantes:
SELECT nome, vencimento, 
       DATEDIFF(vencimento, CURDATE()) AS dias_restantes
FROM contas_mes;
